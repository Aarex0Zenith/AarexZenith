// roadmap

THE JOURNEY

The honest roadmap from zero to ethical hacker. No shortcuts. No fluff. Just the real path.

STEP 01
LINUX FUNDAMENTALS

Everything in cybersecurity runs on Linux. Master the file system, permissions, bash scripting, process management, and networking commands. Kali Linux is your primary environment.

  • Install Kali Linux in VirtualBox
  • Master core commands: ls, grep, find, chmod, netstat, nmap
  • Write at least 10 bash automation scripts
STEP 02
NETWORKING DEEP DIVE

Without networking knowledge, you are completely blind as a hacker. TCP/IP, OSI model, DNS, HTTP/HTTPS, ARP, and routing are the foundation of every attack and defense.

  • Study CompTIA Network+ as a reference guide
  • Practice live traffic analysis with Wireshark
  • Understand exactly how a website request travels end-to-end
STEP 03
PROGRAMMING

Python is your primary weapon. Start simple — automate tasks, write port scanners, build security tools from scratch. Then learn JavaScript to understand client-side exploitation.

  • Python: requests, socket, subprocess, scapy
  • Build 5 original security tools on GitHub
  • Read and study open source exploit code
STEP 04
WEB SECURITY

The OWASP Top 10 is your technical bible. Learn each vulnerability class in depth — SQLi, XSS, IDOR, SSRF, CSRF, and file upload bypasses. Practice on DVWA and PortSwigger Labs daily.

  • Build a complete Burp Suite workflow
  • Complete all PortSwigger Web Security Academy labs
  • Read every public HackerOne disclosed report you can find
STEP 05
CTF PRACTICE

Capture The Flag competitions are where theory transforms into real skill. Solve challenges across web, crypto, forensics, binary exploitation, and reverse engineering. Lose often. Learn every time.

  • TryHackMe — complete 100 rooms minimum
  • HackTheBox — solve 20+ machines
  • Participate in CTFtime.org events weekly
STEP 06
BUG BOUNTY

Convert your skills into real-world impact and income. Start with private programs, study scope carefully, and report every finding responsibly. One well-written P1 report changes your entire career trajectory.

  • Active accounts on HackerOne, Bugcrowd, Intigriti
  • Focus on one target for a full 30 days
  • Document everything — write professional, reproducible reports
STEP 07
ADVANCED SECURITY RESEARCH

This is the frontier. CVE discovery, malware analysis, red teaming, cloud security, and binary exploitation. Build original tools, publish research, and create a reputation that speaks for itself.

  • Target certifications: OSCP, CEH, eJPT
  • Build a public GitHub security portfolio
  • Write security blog posts, publish writeups, contribute to open source
